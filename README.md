# 多項式シミュレーター & 可視化ツール (Polynomial Simulator & Visualizer)

Node.js環境で動作する、多項式の計算・評価、およびその挙動をCSVやグラフ画像（PNG）として出力するシミュレーターです。外部ライブラリに頼らない数学的な多項式オブジェクトの操作と、Chart.jsを使用した高度な可視化を両立しています。

## 🚀 機能特徴

- **多項式の数学的演算**: 加算、乗算、微分（導関数の作成）をサポート。
- **ホーナー法（Horner's method）による高速評価**: 任意の $x$ に対する $y$ の値を効率的に計算。
- **シミュレーションデータのCSV出力**: 表計算ソフト（ExcelやGoogleスプレッドシート）で即座に解析可能。
- **サーバーサイドでのグラフ生成**: ブラウザを起動することなく、Node.js単体で美しい放物線や3次関数などのグラフ画像（PNG）を出力。

---

## 🛠 動作環境・前提条件

- **Node.js**: v16 以上推奨
- **OS**: Windows / macOS / Linux
  ※グラフ生成ライブラリ（`canvas`）の内部依存関係上、OSによってはビルドツール（PythonやC++コンパイラ）が必要になる場合があります。
