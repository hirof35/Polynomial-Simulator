## 🚀 機能特徴

- **多項式の数学的演算**: 加算、乗算、微分（導関数の作成）をサポート。
- **ホーナー法（Horner's method）による高速評価**: 任意の $x$ に対する $y$ の値を効率的に計算。
- **シミュレーションデータのCSV出力**: 表計算ソフト（ExcelやGoogleスプレッドシート）で即座に解析可能。
- **サーバーサイドでのグラフ生成**: ブラウザを起動することなく、Node.js単体で美しい放物線や3次関数などのグラフ画像（PNG）を出力。

---

## 🛠 動作環境・前提条件

- **Node.js**: v16 以上推奨
- **OS**: Windows / macOS / Linux
  ※グラフ生成ライブラリ（`canvas`）の内部依存関係上、OSによってはビルドツール（PythonやC++コンパイラ）が必要になる場合があります。

---

## 📦 インストール方法

1. このリポジトリをクローンまたはファイルのダウンロードをします。
2. プロジェクトのルートディレクトリで以下のコマンドを実行し、必要な依存パッケージをインストールします。

```bash
npm install
主要な依存パッケージ:chart.js: グラフ描画エンジンchartjs-node-canvas: Node.js環境でChart.jsを動かすためのラッパー💻 使い方・実行方法1. シミュレーションの実行メインスクリプトを実行すると、指定された多項式（初期設定では $3x^2 + 2x + 1$）のシミュレーションが走り、データと画像が自動生成されます。Bashnode visualizer.js
2. 出力される成果物実行が成功すると、プロジェクトのルートディレクトリに以下の2つのファイルが生成されます。output.csv: 指定した範囲（例: $-2.0 \le x \le 2.0$、刻み幅 0.1）のデータ点一覧。graph.png: 計算された多項式の形状を可視化した線グラフ画像。📐 コードの拡張方法（カスタマイズ）visualizer.js 内のパラメータを書き換えることで、自由な多項式や範囲でシミュレーションが可能です。多項式の変更係数の配列を書き換えます。インデックスがそのまま「次数」に対応します。JavaScript// P(x) = 5x^3 - 2x + 4 を定義したい場合
